# Quick Start Guide - Legal Case Manager

## Installation (First Time Only)

### Windows:
1. Double-click `install.bat`
2. Wait for installation to complete

### macOS/Linux:
1. Open Terminal in this folder
2. Run: `./install.sh`
3. Wait for installation to complete

## Running the Application

### Every Time You Want to Use It:
1. Open Terminal/Command Prompt in this folder
2. Run: `python app.py`
3. Open your browser and go to: http://127.0.0.1:5000

### To Stop the Application:
- Press `CTRL+C` in the terminal

## First Steps

1. **Dashboard** - See overview of your practice
2. **Cases** - Click "Add New Case" to create your first case
3. **Hearings** - Click "Schedule Hearing" to add upcoming dates

## Important Files

- `app.py` - Main application (don't delete!)
- `legal_case_manager.db` - Your database (BACKUP THIS FILE!)
- `templates/` - Page templates
- `static/` - Styling files

## Backup Your Data

**IMPORTANT**: Regularly backup `legal_case_manager.db` file!

Simply copy this file to a safe location (Google Drive, USB drive, etc.)
