## 🎯 Quick Start Checklist

### First Time Setup:
1. ✅ Extract all files to a folder
2. ✅ Run the installation script:
   - Windows: Double-click `install.bat`
   - Mac/Linux: Run `./install.sh`
3. ✅ Wait for Flask to install
4. ✅ Run: `python app.py`
5. ✅ Open browser to: http://127.0.0.1:5000

### Every Time You Use It:
1. ✅ Open terminal in the application folder
2. ✅ Run: `python app.py`
3. ✅ Open browser to: http://127.0.0.1:5000
4. ✅ When done, press CTRL+C to stop

## 🆘 Getting Help

**Before You Start:**
1. Read QUICK_START.md
2. Follow installation steps carefully
3. Ensure Python is installed

**If You Have Issues:**
1. Check README.md troubleshooting section
2. Verify Python version: `python --version`
3. Verify Flask installed: `pip list | grep Flask`
4. Check terminal for error messages

**Common Issues:**
- Port 5000 already in use → Change port in app.py
- Can't access application → Check firewall
- Database errors → Delete .db file to start fresh

## 🔄 Updates & Backups

### Backing Up Your Data:
1. Close the application (CTRL+C)
2. Copy `legal_case_manager.db` file
3. Store in safe location (cloud, USB, etc.)
4. Date the backup file

### Restoring Data:
1. Close the application
2. Replace `legal_case_manager.db` with backup
3. Restart application

### Future Updates:
- Keep your database file safe
- Replace all files EXCEPT the .db file
- Your data will remain intact
